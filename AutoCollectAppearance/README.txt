AutoCollectAppearance v6.0 (Ascension) - Automatically accepts appearance collection popups

Changes in v6.0:
- Fixed for Ascension/Wrath 3.3.5 servers.
- Checks all active static popup frames dynamically.
- Automatically clicks Accept when popup text contains:
  "Are you sure you want to collect the appearance of"
- Works even if the popup appears with a slight delay.
- added a clickable button for bulk collect

Installation:
1. Unzip into Interface\AddOns\AutoCollectAppearance
2. Enable the addon in-game.
3. Done! Appearance collection confirmations will now auto-accept.
4. Interface > addons > AutoCollectAppearance for scale and customizable text
